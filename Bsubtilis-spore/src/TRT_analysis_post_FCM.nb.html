d.clean <-
   d%>%
   # make spore NAs into 1 (0 on log scale)
   mutate(spore = if_else(!is.na(spore), 1, spore))%>%
   # make spore counts lower than 100 events 1 (0 on log scale)
   mutate(spore = if_else(spore<100, 1, spore) )%>%
   # make veg counts lower than 100 events 1 (0 on log scale)
   mutate(veg = if_else(veg<100, 1, veg) ) %>% 
  # unique sample name
  mutate(unq.sample = str_c(t.exp, bio.rep, exp.plate,trt, sep = "_")) %>% 
  # blanks are not useful for this analysis
  filter(bio.rep!="blank") %>% 
  
  mutate(spore = if_else(is.na(spore), 0.1, spore),
         spore.ml = if_else(is.na(spore.ml), 0.1, spore.ml))

Replication outliers

I have made 3 technical replicate measurements in the flow-cytometer for each sample. There are some measurements which are obviously way off, e.g. in “WED12” samples some wells did not get SYBR to to pipetting error. To get rid of these I will choose from each triplicate the 2 points that are in best agreement and remove the 3rd point. There is a function to idenify such a point in pacakge ‘outliers’ > outlier {outliers} Finds value with largest difference between it and sample mean, which can be an outlier.

Gompertz <- function(x, y0, ymax, k, lag){
      result <- y0 + (ymax -y0)*exp(-exp(k*(lag-x)/(ymax-y0) + 1) )
      return(result)
      }
